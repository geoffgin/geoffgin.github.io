Projects

Generative AI SQL Chatbot

Built a chatbot that converts natural language into SQL using LLMs + RAG/TAG techniques. Containerized the system and deployed a Python-based UI with Streamlit and Docker. Integrated explainability features to track cost and query logic. Use case: enhancing data access for non-technical stakeholders.

SIADS Milestone II: Exploring the Impact of Pre-Existing Health Conditions on Various Diseases

This project focus' on cardiovascular mortality. It explores the role of pre-existing conditions and their correlation with cardiovascular and COVID-19 mortality. Utilized the CDC's Behavioral Risk Factor Surveillance System (BRFSS) as the primary dataset. Employed regression, random forest, and K-Nearest Neighbors models for analysis. Unsupervised analysis involved PCA and k-means clustering to identify natural groupings of conditions.
